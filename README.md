# 🧮 Tkinter Calculator (Python)

A simple graphical calculator built using Python’s built-in **Tkinter** library. This project demonstrates how to create a functional GUI application with basic arithmetic operations.

---

## 📌 Features

- User-friendly graphical interface  
- Supports basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear button to reset input  
- Real-time input display  
- Error handling for invalid operations (e.g., division by zero)

---

## 🛠️ Technologies Used

- Python 3  
- Tkinter (standard GUI library included with Python)
